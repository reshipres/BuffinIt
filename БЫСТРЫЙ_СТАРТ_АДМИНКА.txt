📝 КАК ДОБАВИТЬ СЕБЯ АДМИНОМ:

1. Узнайте свой User ID:
   • Напишите боту: @userinfobot
   • Он пришлет ваш ID (например: 123456789)

2. Откройте файл: admins.py

3. Добавьте себя в список ADMINS:
   
   ADMINS = [
       (123456789, "ваш_username"),
   ]

4. Сохраните файл и запустите бота

═══════════════════════════════════════════════════════════════

🚀 КАК ИСПОЛЬЗОВАТЬ АДМИН-ПАНЕЛЬ:

1. Запустите бота
2. Отправьте команду: /admin
3. Вы увидите меню с кнопками:

   📊 Статистика         - сколько заявок, пользователей
   📋 Все заявки         - последние 10 заявок с деталями
   👥 Список админов     - кто имеет доступ к админке
   ℹ️ О боте            - информация о версии и функциях

═══════════════════════════════════════════════════════════════

💡 ПРИМЕРЫ:

Один админ:
ADMINS = [
    (123456789, "admin_username"),
]

Несколько админов:
ADMINS = [
    (123456789, "first_admin"),
    (987654321, "second_admin"),
    (555555555, "third_admin"),
]

═══════════════════════════════════════════════════════════════

⚠️ ВАЖНО:

• НЕ забудьте запятую после каждой строки
• Username указываем БЕЗ знака @
• После изменений в admins.py - перезапустите бота
• Если не работает - проверьте логи бота
